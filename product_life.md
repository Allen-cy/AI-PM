## 2026-05-14T13:45:00+08:00 - 新增功能：项目全流程交付管理蓝图 V3

### 任务元数据
- **指令类型**：新增
- **时间戳**：2026-05-14T13:45:00+08:00
- **关联版本号**：v3.0

### 逻辑链路
- **AI 思考过程**：用户提供了蓝图业务模块的文本结构，要求完美复刻图片样式并确保可修改。分析需求后，决定采用水平泳道图（Swimlane Flowchart）的设计模式来展示业务场景，使用 TailwindCSS 构建精美的卡片和连线效果。同时为了满足可修改的要求，引入了双击切换 `contentEditable/input` 的 React State 内联编辑模式，满足用户在不用第三方库的条件下直接修改文字的需求。
- **选用的关键调优解决方案**：使用 React 的 `useState` 结合内联 `input` 构建节点内容的双击编辑功能；设计专门的“贯穿全流程”模块横跨项目管理层级。

### 执行细节
- **调用的具体工具**：`list_dir`, `view_file` (分析已有页面), `write_to_file` (创建 `src/app/blueprint-v3/page.tsx`), `replace_file_content` (更新首页路由)。
- **遇到的技术阻碍**：由于无法直接查看用户的图片附件，采用了标准的 UI 最佳实践实现了用户文字描述的泳道结构，并在页面顶端预留了带容错提示的 `<img>` 占位，指导用户放置附件原图。
- **最终处理方案**：成功创建 `blueprint-v3` 页面并嵌入首页。

### 未来演进
- 考虑引入图形化连线库（如 React Flow）来实现更自由的拖拽编辑。
- 增加将修改后的蓝图导出为图片或 PDF 的功能。

### 阶段总结
完成了 V3 蓝图视图的开发，包含了图片附件展示区域与可交互的文本编辑流程图，与用户的具体字段需求完全对应。

## 2026-06-22 - V4.0：从Mock问答升级为可追溯RAG

- **版本：** 4.0.0
- **用户变化：** 知识问答显示真实知识页、来源ID、相关度和实际检索模式；证据不足或需要飞书实时数据时明确拒答。
- **技术变化：** 新增本地知识Provider、10篇已审快照、查询/健康API、权限过滤、RFC 9457错误与审计ID。
- **兼容性：** 旧`/api/knowledge`入口保留，但已移除Mock生成器。
- **验证：** 20个自动化测试；20题黄金评测；生产构建和目标文件Lint作为发布门禁。
- **后续：** 飞书实时业务数据、幂等同步账本和向量Provider。

### V4.0 飞书增量

- 建立项目、里程碑、任务、风险、合同、回款、成本和同步账本8张真实Base表。
- Bot创建后自动授予当前用户可管理权限。
- 新增服务端飞书应用身份客户端与连接健康接口，默认关闭失败且不暴露密钥。
- 在用户登录和行级授权完成前，不开放项目记录公共读取接口。

## 2026-06-22 - V5.0：RAG与飞书协作闭环

- **版本：** 5.0.0
- **用户变化：** 知识问答扩展到27篇审定知识；WBS、SPI/CPI、方法论组合和系统事实边界可直接带引用回答。
- **检索变化：** 生产端升级为确定性的词法与元数据加权检索，30题黄金集Top1全通过；健康接口保持0向量的真实边界。
- **飞书变化：** 新增事件回调、持久化幂等、消息、任务、日历与结构化文档动作。

- **恢复能力：** 失败动作进入同步账本并允许最多3次重试；参会人写入失败会回滚空日程。
- **验证：** 41项测试覆盖RAG、回调、鉴权、账本、四类动作和30题黄金集。
- **后续：** 完成生产环境变量、飞书后台回调、Vercel部署与用户级行权限查询。
